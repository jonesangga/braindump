# Prolog

## IO

```prolog
write/1.
write(real).
write('real').
write(2).
write([1,2,'a']).

`nl/0`.

read/1.
read(X).
|: real.
X = real.

put/1
put(97).
a

get0/1
get0(N).
|: a
N = 97

get/1
same as get but ignoring left white space chars
```
